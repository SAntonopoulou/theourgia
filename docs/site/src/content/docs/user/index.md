This section will hold user documentation for working magicians.

Theourgia is currently in **planning phase**; no user-facing application exists yet. As features land in subsequent phases, this guide will grow with them — user docs ship in the same PR as the feature they document (per the project's documentation discipline).

When the platform is functional, you can expect coverage here of:

- Setting up your first vault
- Multi-calendar overlays and astrology configuration
- Writing magical record entries
- Tarot and other divination workbenches
- Sigil generation
- The entity ledger
- Joining and participating in networks
- Publishing and newsletters
- Privacy and encryption modes

For now, see the [Status & roadmap](/start/status/).
