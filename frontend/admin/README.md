# Theourgia — Admin SPA

React 19 single-page application serving the magician's vault dashboard:

- Home / today (multi-calendar widget, planetary hour, recent entries, quick capture)
- Journal index and entry editor (Tiptap)
- Entity ledger
- Library catalog
- Divination workbench
- Sigil studio, magical circle builder, talisman designer
- Analytics ("scientific illuminism")
- Settings (account, security, encryption mode, networks, plugins, theme)

## Status

**Planning phase — empty placeholder package.** Implementation begins with Phase 02 (Frontend Foundations).

## Development

From the **repository root**:

```bash
pnpm --filter @theourgia/admin dev
```
