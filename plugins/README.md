# Theourgia — Reference Plugins

This directory contains plugins shipped as part of Theourgia (first-party reference implementations).

Third-party plugins live in their own repositories and are distributed via the [official Theourgia plugin registry](../FEATURES.md#17-plugin-ecosystem).

## Status

**Planning phase — no plugins yet.** Implementation begins with Phase 14 (Plugin Ecosystem) — see [../plan/14-plugin-ecosystem.md](../plan/14-plugin-ecosystem.md).

## Planned reference plugins

Per [FEATURES.md §17](../FEATURES.md):

- `theourgia-plugin-runes-extended` — Younger Futhark, Anglo-Saxon Futhorc, bind-rune designer
- `theourgia-plugin-egyptian-decans` — Egyptian decanic chart layer
- `theourgia-plugin-correspondences-777` — Liber 777 correspondence table importer
- `theourgia-plugin-import-day-one` — import from Day One journal exports
- `theourgia-plugin-export-obsidian` — export to Obsidian-compatible markdown vault
- `theourgia-plugin-notification-matrix` — send reminders via Matrix
- `theourgia-plugin-divination-tea-leaves` — non-mechanical divination plugin demo

Each reference plugin lives in its own subdirectory once authored.
