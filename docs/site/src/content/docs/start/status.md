Theourgia is in **planning phase**. No runnable code yet.

The project is being built in **17 architecturally-ordered phases**. Each phase is complete when it meets its Definition of Done (tests, docs, security review, migration story).

## Pace

The project is intentionally not in a rush. Quality, maintainability, and tradition-respectful depth are the priorities — not speed-to-MVP. From the project plan:

> "Theourgia is community infrastructure for practicing magicians. The success metric is **adoption by practitioners**, not revenue."

This page is updated continuously as phase status changes.
